# Nhập môn an toàn thông tin ⚠

## [Deployed here ✅](https://nmattt.noctrlz.tech/)

## Cách cài đặt

- Cài đặt yarn [tại đây](https://yarnpkg.com/)
- Cài đặt các dependency, chạy lệnh
  
  ```
  yarn install
  ```
- Khởi chạy
  ```
  yarn start
  ```
- Truy cập vào đường dẫn http://localhost:3000/
